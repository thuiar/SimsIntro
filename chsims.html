CH-SIMS v2.0: Chinese Fine-grained Multi-label Sentiment Analysis Dataset

Overview

The statistics of CH-SIMS v2.0 is shown in below figure. Specifically, CH-SIMS v2.0 contains 4402 supervised instances, denoted as CH-SIMS v2.0 (s), and 10161 unsupervised instances, denoted as CH-SIMS v2.0 (u). For the supervised resources, the CH-SIMS v2.0 (s) shares similar properties with the original dataset. While, for the unsupervised resources, the CH-SIMS v2.0 (u) shows a much more diverse duration distribution simulating the potential real-world scenarios. In addition, the textual modality of the unsupervised instances is directly collected from the corresponding transcript without being manually refined and thus might contain noise.

The CH-SIMS v2.0: Chinese Fine-grained Multi-label Sentiment Analysis Dataset (CH-SIMS v2.0) dataset is an enhanced and extended version of CH-SIMS. The baseline experiments are conducted using the MMSA platform.

In addition, we split train, valid and test set close to 9:2:3. NEG: Negative, WNEG: Weakly Negative, NEU: Neutral, WPOS: Weakly Positive, POS: Positive. We guarantee that the test set and the train set are speaker-unrelated.
